# Setup

## Prerequisites

Make sure you have the following installed:

- [Git](https://git-scm.com/downloads) - Version control system
- [Make](https://www.gnu.org/software/make/#download) - Build automation tool
- [Docker](https://docs.docker.com/desktop/) - Containerization platform

## Installation

Clone the repository and navigate to the project directory:

```bash
git clone https://github.com/tibia-oce/foundation
cd foundation
```

## Setup

Install dependencies and prepare the project:

```bash
make setup
```

## Next Steps

Once setup is complete, check out the [usage guide](https://tibia-oce.github.io/foundations/usage) to learn how to use the project, or see the [contributing guide](https://tibia-oce.github.io/foundations/contributing) if you'd like to contribute.
